# 文档总索引

这份文件用于给人和自动文档工具提供一个稳定入口。

如果你第一次阅读 Axiom，建议按下面顺序看：

1. `README.md`
2. `docs/ARCHITECTURE.md`
3. `docs/CURRENT_STATE.md`
4. `docs/ARCHITECTURE_MINDMAP.md`

## 文档说明

### README.md

仓库首页。

用于快速说明：

- 项目是什么
- 当前阶段是什么
- 当前架构是什么
- 当前优先级是什么
- 应该先看哪些文档

### docs/ARCHITECTURE.md

架构说明文档。

用于说明：

- Axiom 的长期目标
- v0.1 当前范围
- 当前部署架构
- 存储模型
- API 职责
- 工程边界和开发优先级

### docs/CURRENT_STATE.md

当前状态文档。

用于记录：

- 目前已经实现了什么
- 当前数据快照是什么
- 当前主要风险是什么
- 下一步最应该先修什么

### docs/ARCHITECTURE_MINDMAP.md

可视化架构文档。

用于快速查看：

- v0.1 目标架构
- 当前实现状态
- 当前阶段推进位置
- 近期优先级关系
