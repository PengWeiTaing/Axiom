# Axiom 浏览器扩展

支持 Edge / Chrome / 任何 Chromium 内核浏览器。

## 安装 (Edge)

1. 打开 `edge://extensions/`
2. 打开左下角「开发人员模式」
3. 点击「加载解压缩的扩展」
4. 选择本目录 (`extensions/chrome/`)
5. 点击工具栏上的 🧩 图标，找到 Axiom，点 📌 固定

## 使用

- **点击图标** → 弹出输入框 → 打字 → Enter → AI 自动分类保存
- **📄 当前页** → 保存当前页面标题+链接
- **🔗 抓取链接** → 抓取当前页内容（B站视频可抓字幕）
- **右键菜单** → 选中文字 → 右键 → 保存到 Axiom
