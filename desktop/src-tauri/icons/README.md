# App Icons

Place the following icon files in this directory for Tauri bundling:

- `icon.png` (1024x1024, RGBA PNG — source for all platforms)
- `icon.ico` (Windows, must include 256x256)
- `icon.icns` (macOS)

Run `npx tauri icon path/to/icon.png` to auto-generate all required sizes
from a single 1024x1024 source PNG.
